# fastsw
fastsw is a web proxy powered by service workers, and libcurl.js, this does not need any backend whatsoever, and just needs a Wisp server, this is still in alpha, site support is pretty bad (for now)

Website support

YouTube  (works, videos play fine, may get a login prompt if the wisp servers ip is sending too many requests)

Discord (partical, loging in works so does fine, but Discord might lock you account until a password reset, it only happens pretty rarely thouh)

now.gg (partial)

Google (not yet, working on it)
